# 1. Next.js

**Next.js** is a React framework used to build modern full-stack web applications. It provides powerful features such as:

- File-based Routing
- Server-Side Rendering (SSR)
- Static Site Generation (SSG)
- API Routes
- Dynamic Routing
- SEO Optimization

## Why Next.js?

Next.js improves the development experience by combining frontend and backend capabilities in a single framework.

### Key Benefits

- Better SEO compared to traditional client-side rendering
- Faster page loading
- Built-in routing system
- Full-stack development support
- Optimized performance

## Concepts Learned

- File-based Routing
- Dynamic Routes
- Layouts
- API Routes
- Server Components
- Client Components
- SSR (Server-Side Rendering)
- SSG (Static Site Generation)

## Notes

Next.js allows developers to create highly performant and SEO-friendly applications. It extends React with additional features that simplify modern web development.
